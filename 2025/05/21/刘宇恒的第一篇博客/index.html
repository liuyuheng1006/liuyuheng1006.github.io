第3次作业：MPI编程实验

ZY2406217 刘宇恒

一、实验内容与方法

1. 实验内容

  • 用MPI编程实现矩阵并行乘法程序
  • 矩阵大小≥8000×8000，矩阵元素为双精度浮点数(约需要480MB内存)
  • 编程语言：C/C++
  • 比较并行与串行程序的加速比
  • 注意:
    • 统计计算时间时，应排除准备数据的时间(如果有I/O)
    • 统并行程序的性能可扩展性：线程个数可变、局部性、

2. 实验方法

  • 串行算法：三重循环直接计算，优化循环顺序为 i->k->j 以提高缓存命中率。
  • 并行算法：
    • 数据划分：主进程（rank=0）将矩阵 A 按行分块，通过 MPI_Scatter 分发到各进程。
    • 广播共享数据：矩阵 B 通过 MPI_Bcast 广播到所有进程。
    • 局部计算：各进程计算分块行与矩阵 B 的乘积，结果存入 local_C。
    • 结果收集：通过 MPI_Gather 将各进程的 local_C 汇总到主进程的 C 中。
  • 关键函数：MPI_Scatter, MPI_Bcast, MPI_Gather。

二、实验过程

1. 实验环境

**Intel Xeon Gold 6230R CPU **

  • 核心与线程
    • 核心数: 26核
    • 线程数: 52线程，支持超线程技术，每个核心处理2个线程。
    • 多任务能力: 适合高密度计算任务，如科学模拟、大规模数据处理。
  • 频率与性能
    • 基础频率: 2.10 GHz
    • 最大频率: 4.00 GHz
    • 性能特点: 高核心数适合并行计。

2. 实验步骤

  • init_matrix: 初始化随机矩阵。

    void init_matrix(double *matrix, int N) {
    for (int i = 0; i < N * N; ++i) {
        matrix[i] = (double)rand() / RAND_MAX;
    }
}

  • serial_matmul: 串行矩阵乘法。

    void serial_matmul(double *A, double *B, double *C, int N) {
    for (int i = 0; i < N; ++i) {
        for (int k = 0; k < N; ++k) {
            double a = A[i * N + k];
            for (int j = 0; j < N; ++j) {
                C[i * N + j] += a * B[k * N + j];
            }
        }
    }
}

  • parallel_matmul: MPI并行乘法。

    void parallel_matmul(int rank, int size, double *A, double *B, double *C, int N) {
    int rows_per_proc = N / size;
    double *local_A = (double *)malloc(rows_per_proc * N * sizeof(double));
    double *local_C = (double *)malloc(rows_per_proc * N * sizeof(double));

    MPI_Scatter(A, rows_per_proc * N, MPI_DOUBLE, local_A, rows_per_proc * N, MPI_DOUBLE, 0, MPI_COMM_WORLD);
    MPI_Bcast(B, N * N, MPI_DOUBLE, 0, MPI_COMM_WORLD);

    for (int i = 0; i < rows_per_proc * N; ++i) {
        local_C[i] = 0.0;
    }

    for (int i = 0; i < rows_per_proc; ++i) {
        for (int k = 0; k < N; ++k) {
            double a = local_A[i * N + k];
            for (int j = 0; j < N; ++j) {
                local_C[i * N + j] += a * B[k * N + j];
            }
        }
    }

    MPI_Gather(local_C, rows_per_proc * N, MPI_DOUBLE, C, rows_per_proc * N, MPI_DOUBLE, 0, MPI_COMM_WORLD);

    free(local_A);
    free(local_C);
}

  • main: 控制逻辑与性能统计。

    int main(int argc, char **argv) {
    MPI_Init(&argc, &argv);
    int rank, size;
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Comm_size(MPI_COMM_WORLD, &size);

    const int N = 8000;
    double *A = NULL, *B = (double *)malloc(N * N * sizeof(double)), *C = NULL; 
    double start_time, end_time;

    if (rank == 0) {
        A = (double *)malloc(N * N * sizeof(double));
        C = (double *)malloc(N * N * sizeof(double));
        init_matrix(A, N);
        init_matrix(B, N);
    }

    MPI_Barrier(MPI_COMM_WORLD);

    if (size == 1) {
        if (rank == 0) {
            for (int i = 0; i < N * N; ++i) C[i] = 0.0;
            start_time = MPI_Wtime();
            serial_matmul(A, B, C, N);
            end_time = MPI_Wtime();
            printf("Serial time: %.2f s\n", end_time - start_time);
        }
    } else {
        if (N % size != 0) {
            if (rank == 0) fprintf(stderr, "N must be divisible by size\n");
            MPI_Finalize();
            free(B); 
            return 1;
        }

        if (rank == 0) {
            for (int i = 0; i < N * N; ++i) C[i] = 0.0;
            start_time = MPI_Wtime();
        }

        parallel_matmul(rank, size, A, B, C, N);

        if (rank == 0) {
            end_time = MPI_Wtime();
            printf("Parallel time (%d procs): %.2f s\n", size, end_time - start_time);
        }
    }

    if (rank == 0) {
        free(A);
        free(C);
    }
    free(B);

    MPI_Finalize();
    return 0;
}

  • 程序编译和执行：

    # 编译
mpicc -O3 -o mpi_matmul mpi_matmul.c -lmpi
# 串行模式
mpiexec -n 1 ./mpi_matmul
# 并行模式（4进程）
mpiexec -n 4 ./mpi_matmul

2. 实验结果分析

实验数据验证了以下结论：

  • 并行有效性：8线程实现6.65-7.02倍加速，达到83-88%的并行效率

  • 更大数据集（40,000k）的并行效率更高，猜想可能的原因是：

    • 任务粒度更优，更多子任务分摊同步开销。
    • 缓存利用率提升，L3缓存命中率增加。
